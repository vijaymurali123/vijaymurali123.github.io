# Wedding Website

A beautiful, responsive wedding website created for Vijay's wedding celebration.

## Features

- **Hero Section**: Elegant landing page with couple's names and wedding date
- **Navigation**: Smooth scrolling navigation menu
- **Our Story**: Share your love story with guests
- **Wedding Details**: Display ceremony date, time, location, and dress code
- **Photo Gallery**: Showcase memorable moments
- **RSVP Form**: Collect guest responses and messages
- **Responsive Design**: Works perfectly on all devices

## How to Use

1. **Customize Your Information**:
   - Open `index.html`
   - Replace `[Partner Name]` with your partner's name
   - Update `[Wedding Date]`, `[Venue Name]`, `[Venue Address]`, etc.
   - Modify "Our Story" section with your personal story

2. **Add Your Photos**:
   - Replace placeholder images in the gallery section
   - Update image paths in `index.html` (look for `<img src="...">` tags)
   - Recommended image size: 400x300 pixels or similar aspect ratio

3. **Customize Colors**:
   - Edit `styles.css` to change the color scheme
   - Main purple color: `#8b5a7f` - search and replace with your preferred color

4. **Optional Features**:
   - Uncomment the countdown timer code in `script.js` and set your wedding date
   - Integrate RSVP form with a backend service (Google Forms, Formspree, EmailJS)

## Viewing the Website

Simply open `index.html` in any web browser to view your wedding website.

## Deployment

You can host this website for free on:
- **GitHub Pages**: Push to GitHub and enable Pages
- **Netlify**: Drag and drop your folder
- **Vercel**: Connect your repository
- **Firebase Hosting**: Deploy with Firebase CLI
